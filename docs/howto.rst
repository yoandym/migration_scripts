HowTo
=====

Instalacion
-----------

Para instalar el modulo, siga los siguientes pasos:

1. Descargue el modulo desde el repositorio de Github.
2. Copie el modulo en la carpeta de addons de su instancia Odoo.




Cómo usar el modulo para migrar datos
-------------------------------------

A continuacion se muestra un ejemplo de migración de datos entre instancias Odoo:

1. Defina los datos de autenticacion para la instancia de origen y destino.

2. Seleccione el modelo de datos que desea migrar.

3. Cree el mapeo de campos entre las instancias de origen y destiono.

4. Ejecute el proceso de migración.
