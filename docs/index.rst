
Documentation
=============

Module for partial data migration between Odoo instances.

If you want to do a real / full odoo database migration, you should check:
   - The official odoo migration service ( if you have an enterprise licence ) at https://upgrade.odoo.com
   - The OCA openupgrade project at https://github.com/OCA/OpenUpgrade

If you just wanna migrate a couple of models, you can use this module to do it.
