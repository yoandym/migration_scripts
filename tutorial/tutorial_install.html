Instalación#

Instalacion manual#

Puede usar el modulo descargando el codigo fuente desde github, para ello siga los siguientes pasos:

  1. Descargue el modulo desde el repositorio de Github con un cliente git.

git clone https://github.com/yoandym/migration_scripts.git ./migration_scripts

  1. Instale las dependencias del modulo. Es recomendable usar un ambiente virtual para instalar las dependencias.

pip install -r migration_scripts/requirements.txt

Instalacion con pip#

  1. Instale directamente el modulo usando pip. Es recomendable usar un ambiente virtual.

pip https://github.com/yoandym/migration_scripts/archive/refs/heads/main.zip
